# Screenshots

This folder contains screenshots for the README documentation.

To add screenshots:
1. Take screenshots of your running application
2. Name them according to the README format:
   - 1-wallet-disconnected.png
   - 2-wallet-connected.png
   - 3-balance-displayed.png
   - 4-transaction-form.png
   - 5-transaction-processing.png
   - 6-successful-transaction.png
   - 7-transaction-result.png
3. Place them in this folder

Recommended screenshot sizes:
- Width: 800-1200px
- Format: PNG or JPG
- Should clearly show the UI state being documented
